# Smart Shuttle App Design
 SmartShuttle is a conceptual mobile application designed to improve campus transportation at Universiti Teknologi Malaysia (UTM). The project focuses on system analysis, requirements gathering, and user interface design, providing features such as real-time shuttle tracking, route information, and schedule management. Developed using Figma, the project demonstrates my skills in software analysis, UI/UX design, and problem-solving for smart campus solutions.
